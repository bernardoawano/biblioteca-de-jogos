# 📚Biblioteca de Jogos


Uma biblioteca de jogos feita com React e utilizando o Vite

## 🛠️Funcionalidades

- Criar uma bibloteca virtual
- Armazenar itens do seu interesse
- Remover itens que não lhe interessam mais
